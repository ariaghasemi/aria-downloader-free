# 📦 IR-Downloader

یک ابزار قدرتمند مبتنی بر GitHub Actions برای دانلود فایل‌ها از لینک مستقیم، **دسته‌بندی خودکار**، **رمزگذاری** و **ذخیره‌سازی دائمی در مخزن گیت‌هاب**. دیگر نگران حذف خودکار فایل‌ها یا محدودیت حجم نباشید.

---

## ✨ قابلیت‌های کلیدی

- 🔗 **دانلود از هر لینک مستقیم** (HTTP/HTTPS)
- 🧠 **تشخیص خودکار نوع فایل** → ویدئو / صدا / تصویر / سند / سایر
- 📁 **ساختار هوشمند پوشه‌ها**:  
  `downloads/{دسته}/{نام انتخابی شما}/` همراه با فایل‌های 7z تقسیم شده
- 🔐 **رمزگذاری دلخواه** (اختیاری) با 7-Zip
- ✂️ **تقسیم به تکه‌های دلخواه** (پیش‌فرض 95 مگابایت)
- 💾 **ذخیره دائمی در خود مخزن** (نه Artifact موقتی)
- 📥 **دانلود یکجا** با کلیک روی `Code → Download ZIP`
- 🔄 **عدم تداخل فایل‌های قبلی** – هر فایل در پوشه جداگانه
- ⏱️ **بدون محدودیت زمانی** (برخلاف Artifactهای 90 روزه)

---

## 🆕 ویژگی‌های اضافه شده در نسخه IR-Downloader

نسخه فعلی نسبت به نسخه پایه، بهبودهای چشمگیری دارد:

### 1. 🏷️ **تعیین نام دلخواه برای فایل**
- **نسخه پایه:** نام فایل به‌طور خودکار از لینک استخراج می‌شد
- **نسخه IR-Downloader:** شما می‌توانید نام دلخواه خود را وارد کنید. این نام هم برای نام پوشه و هم برای نام فایل نهایی استفاده می‌شود.

### 2. 🗂️ **ساختار پوشه‌بندی شفاف‌تر**
- **نسخه پایه:** `downloads/دسته/نام_اتوماتیک/archive.7z.00x`
- **نسخه IR-Downloader:** `downloads/دسته/نام_انتخابی_شما/نام_انتخابی_شما.7z.00x`

### 3. 🎯 **نام‌گذاری معنادار فایل‌های خروجی**
- فایل‌های 7z تولید شده با همان نام انتخابی شما ذخیره می‌شوند
- دیگر خبری از نام‌های گنگ `archive.7z` نیست

### 4. 📛 **پاکسازی خودکار کاراکترهای غیرمجاز**
- نام‌های وارد شده به‌طور خودکار از کاراکترهای مضر پاکسازی می‌شوند (`\/:*?"<>|`)
- جلوگیری از ایجاد خطا در سیستم فایل گیت‌هاب

### 5. 📋 **پیام کامیت خوانا و قابل فهم**
- کامیت‌ها با فرمت: `Add: [نام انتخابی] (category: [نوع])`
- رهگیری آسان فایل‌های ذخیره شده در تاریخچه

### 6. 🎨 **ورودی کاربرپسندتر**
- فیلد جداگانه و اجباری برای نام فایل
- راهنمای واضح در توضیحات فیلدها

## 🚀 نحوه استفاده

خیلی ساده‌تر از چیزی‌ست که فکر می‌کنی:

### ۱. پروژه را فورک (Fork) کن
- دکمه **Fork** (بالای صفحه، سمت راست) رو بزن
- دکمه سبز **Create fork** رو بزن
- یک کپی کامل از پروژه توی اکانت خودت داری

### ۲. دسترسی نوشتن را فعال کن (فقط یک بار!)
- در ریپازیتوری فورک شده، به **Settings** برو
- از منوی چپ: **Actions → General**
- برو به بخش **Workflow permissions**
- گزینه **Read and write permissions** رو انتخاب کن
- دکمه **Save** رو بزن

### ۳. استفاده کن!
- به تب **Actions** در مخزن خودت برو
- روی **IR-Downloader** در سمت چپ کلیک کن
- دکمه **Run workflow** رو بزن
- اطلاعات زیر رو وارد کن:
  - 🔗 **لینک مستقیم فایل** (اجباری)
  - 📛 **نام فایل** (اجباری) - مثل `فیلم_آموزشی` یا `پروژه_۱۴۰۳`
  - 🔐 **رمز فایل فشرده** (اختیاری)
  - 📦 **حجم هر تکه** (اختیاری، پیش‌فرض `95m`)
- دکمه سبز **Run workflow** رو بزن

## 💡 نکات مهم

- پس از اجرا، کمی صبر کن تا فرآیند دانلود و آپلود کامل بشه
- فایل‌ها به صورت **7z با پسوند 001، 002، ...** ذخیره می‌شوند
- برای استخراج نیاز به **7-Zip** یا **WinRAR** داری
- اگر رمز گذاشتی، هنگام استخراج باید رمز رو وارد کنی
- حجم تکه‌ها رو می‌تونی با فرمت `m` (مگابایت) یا `g` (گیگابایت) مشخص کنی
